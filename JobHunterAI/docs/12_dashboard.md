# 12 - Streamlit Dashboard

## UI View Specifications
The Streamlit application (`app.py`) provides an interactive interface for managing your career pipeline:

- **CRM Analytics Panel**: Visualizes application funnel status distribution and template conversion stats.
- **Wizard Gate**: Evaluates discovered job postings, runs match algorithms, displays tailored resume diffs, and compiles PDFs.
- **Knowledge Graph Explorer**: Browses candidate experiences, projects, and linked evidence items.
- **Recruiter CRM Manager**: Logs conversation histories and messages.
- **Offer Evaluator**: Input details (base salary, equity, work policy) to calculate comparative rankings.
