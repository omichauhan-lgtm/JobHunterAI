# 11 - Interview Preparation Engine

## Prep Packet Outputs
When the pipeline status advances to 'Interview', the prep engine compiles a customized study guide containing:

- **Company Profile Summary**: Products, news, and stack.
- **Likely Tech Questions**: Based on the job description requirements.
- **STAR Behavioral Stories**: Automatically mapped from the candidate's database star stories.
- **System Design Guidelines**: High-level design topics mapped to the tech stack (e.g. FastAPI middleware, database indices).

## Factual Alignment
The preparation packet aligns star stories with actual database evidence (PRs, commits) so the candidate can reference real data.
