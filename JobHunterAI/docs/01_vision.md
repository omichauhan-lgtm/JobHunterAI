# 01 - Platform Vision & Core Philosophy

## Core Philosophy
JobHunterAI is a high-precision **Career Intelligence Platform** that transitions the career search from a volume-based auto-apply approach to a highly targeted, data-driven sales funnel. 

### Core Principles
- **Factual Integrity**: Never invent experience. The system treats the candidate's career data as a single source of truth.
- **Traceability**: Every generated resume bullet must be linked to verified evidence (PRs, commits, metrics).
- **Human-in-the-Loop**: Automation prepares materials, but a human must review and approve before submission.
- **Continuous Learning**: Funnel outcomes feed back into the system to improve project rankings and layout effectiveness.
