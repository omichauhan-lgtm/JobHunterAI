The platform prioritizes matching and tailoring for the following technical roles:
- **Founding Engineer**
- **Backend Engineer (Python/FastAPI)**
- **AI Engineer / AI Infrastructure Engineer**
- **Full Stack Engineer**
- **Software Engineer / Systems Engineer**
- **Technical Implementation Engineer / Solutions Engineer**

Unrelated fields (e.g. SDR, generic sales, marketing, operations) are automatically filtered out.

## 🏢 Startup Stage Verticals & Focus
1. **Early Stage / YC Seed**: Emphasize speed, full-stack prototyping speed, open-source familiarities, and rapid API execution.
2. **Growth / Series A-B**: Emphasize performance, system latency optimizations (e.g. OMI's 45ms latencies), test coverages, and database scaling.
3. **Enterprise**: Emphasize security, multi-tenant schemas, compliance guidelines, and clean design patterns.
