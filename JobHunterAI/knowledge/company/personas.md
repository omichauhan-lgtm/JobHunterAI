# Company Personas & Profiles

## Startup Stage Verticals
1. **Early Stage / YC Seed**: Emphasize speed, full-stack capability, open-source familiarity, and rapid feature execution.
2. **Growth / Series A-B**: Emphasize scalability, system latency optimizations, test coverages, and database schema performance.
3. **Enterprise**: Emphasize security compliance, clean code architecture, structural standards, and cross-team collaborations.
